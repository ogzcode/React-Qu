# Quiz Uygulaması
React.js ve Open Trivia DB altyapısından çektiğim soruları kullanarak geliştirdiğim Quiz uygulaması.

## Kullandığım teknolojiler
* Axios
* html-entity
* React
* Pure-Css
* useEffect

### Kullandığım API adresi
* https://opentdb.com/api_config.php
